# <i class="fas fa-puzzle-piece"></i> Compatibility

## Raspberry Pi Compatibility

The Tentacle T3 is compatible with:
* Raspberry Pi 2
* Raspberry Pi 3
* Raspberry Pi 3 B+
* Raspberry Pi ZERO
* Raspberry Pi ZERO W

## EZO Device Compatibility

### Compatible Devices
* `EZO pH` Circuit
* `EZO EC` Circuit
* `EZO ORP` Circuit
* `EZO DO` Circuit
* `EZO RTD` Circuit
* `EZO PMP` Dosing Pump

### Incompatible circuits
* EZO Flow Meter Totalizer
* Legacy circuits V3.0 - V6.0
